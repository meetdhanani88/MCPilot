# Company Chatbot — Agent + MCP + LLM

A ChatGPT-like assistant that answers questions using your company's database and documents. Built with Express.js, OpenAI, and a simple agent loop — no LangChain needed.

## Architecture

```
User → Chat UI → Express API → Agent Loop → LLM (GPT-4o)
                                    ↕
                              MCP Tool Servers
                            ┌───────┴───────┐
                        Database MCP    Documents MCP
                        (SQLite)        (File Search)
```

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Set up your API key
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY

# 3. Seed sample data
node scripts/seed.js

# 4. Start the server
npm start
```

Open http://localhost:3000 in your browser.

```
│   └── index.html         # Chat UI

│   ├── company.db         # SQLite database
│   └── docs/              # Company documents
└── scripts/
    └── seed.js            # Seed sample data
```

## How It Works

1. User sends a message via the chat UI
2. The **Agent** sends the message + available tool definitions to the **LLM**
3. The **LLM** decides which tools to call (or answers directly)
4. The **Agent** executes tool calls against **MCP servers** (database queries, doc search)
5. Results are fed back to the **LLM** to generate a final answer
6. The answer is streamed back to the user

## Adding Your Own Data

### Database
Edit `scripts/seed.js` to add your own tables and data, then re-run `node scripts/seed.js`.

### Documents
Drop `.txt` or `.md` files into `data/docs/`. The document MCP server picks them up automatically.

### New MCP Server
1. Create a new file in `src/mcp/` (e.g., `api.js`)
2. Export `toolDefinitions` and `executors` (same pattern as `database.js`)
3. Import it in `src/mcp/registry.js`
